## 📌 About the Project

This portfolio was built to present my skills, projects, experience, and certifications in a clean, professional, and visually appealing way. It reflects my transition from System Administration at Keltron to Data Science, with a strong focus on **discipline, production-grade projects, and continuous learning**.

The design follows a premium dark theme with cyan/blue accents, smooth animations, and excellent user experience.

- **HTML5** + **Tailwind CSS**
- **JavaScript** (for animations & interactivity)
- **Responsive Design** (Mobile-first)
- Hosted on **GitHub Pages**
